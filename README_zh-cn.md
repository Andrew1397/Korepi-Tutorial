## ❓ 对于Fans（一定要是Fans身份组）

### 发布

1. 前往 [releases page](https://github.com/Cotton-Buds/colorpicker/releases)
2. 下载`F`版本 (ex. [F8](https://github.com/Cotton-Buds/colorpicker/releases/tag/F8))

### 使用方法

1. 运行 `injector.exe`. 
2. 选择 `Yuanshen.exe` 或者 `GenshinImpact.exe`(检查`cfg.ini`查看 injector 是否选择了正确的游戏路径) 
3. 转到 [我们的 Discord 频道]() `micah-bot-verify` 频道并输入 `/getkey` 来获取密钥。

## ❓ MICAH 构建的版本教程

### 发布

1. 前往 [releases page](https://github.com/Cotton-Buds/calculator/releases)
2. 下载`v0.x.x` 版本 (ex. [v0.9.7](https://github.com/Cotton-Buds/calculator/releases/tag/v0.9.7))

### Usage

1. 运行 `v0.x.x.exe`.
2. 前往 [我们的 discord频道]() `micah-bot-verify`频道并输入`/verify`，然后粘贴**控制台**中显示的“HWID”以获取**密钥文件**.

<a href="#"><img width="965" height="415" src="https://media.discordapp.net/attachments/1126893908597669989/1138425541838516294/Untitled.png"></a>
<a href="#"><img width="1324" height="320" src="https://media.discordapp.net/attachments/1126893908597669989/1138426358217842858/Untitled.png"></a>

3. 下载**密钥文件**并将其放入文件夹中，确保`enc.json`与`v0.x.x.exe`位于同一文件夹中
4. 再次运行`v0.x.x.exe`.
5. 选择`GenshinImpact.exe`或`YuanShen.exe`. （检查`cfg.ini`以查看注入器是否选择了正确的游戏路径）
6. 游戏会自动启动，等待界面出现。
7. 按“TAB”打开 [Korepi](https://github.com/Korepi/Korepi) GUI菜单.

<a href="#"><img width="270" height="200" src="https://images.drivereasy.com/wp-content/uploads/2018/09/img_5ba9fcbbcb694.png"></a>
