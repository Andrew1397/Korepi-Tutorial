## ❓ Fans အတွက် (fan role လိုအပ်သည်)

### Release

1. [Release စာမျက်နှာ](https://github.com/Cotton-Buds/colorpicker/releases)သို့သွားပါ
2. `F` release ကို download လုပ်ပါ (ex. [F8](https://github.com/Cotton-Buds/colorpicker/releases/tag/F8))

### အသုံးပြုနည်း

1. `injector.exe` ကို run ပါ။
2. ``GenshinImpact.exe`` သို့မဟုတ် ``YuanShen.exe`` ကိုရွေးချယ်ပါ
3. ``micah-bot-verify`` channel သို့သွားပြီး key တစ်ခုရရန် ``/getkey`` ကိုရိုက်ထည့်ပါ

<a href="#"><img width="300" height="200" src="https://cdn.discordapp.com/attachments/1126893908597669989/1128329159559622676/image.png"></a>
<a href="#"><img width="365" height="164" src="https://media.discordapp.net/attachments/1126893908597669989/1144891137224613969/image.png"></a>

4. ရရှိလာသော key အား copyယူ၍ injector ထဲသိုထည့်ပါ
5. ဂိမ်းသည် အလိုအလျောက် စတင်လာမည်။ ဂိမ်းမျက်နှာပြင် ပေါ်လာရန်အတွက် ခဏစောင့်ပါ။
6. [Korepi](https://github.com/Korepi/Korepi) GUI ဖွင့်ရန်အတွက် `TAB` key ကို နှိပ်ပါ။

## ❓ MICAH'S BUILDS Tutorial

### Release

1. [Release စာမျက်နှာ](https://github.com/Cotton-Buds/calculator/releases)သို့သွားပါ
2. `v0.x.x` releases ကို download လုပ်ပါ  (ex. [v0.9.7](https://github.com/Cotton-Buds/calculator/releases/tag/v0.9.7))

### အသုံးပြုနည်း 

1. `v0.x.x.exe`ကို Run ပါ
2. [`micah-bot-verify channel`]() သိုသွားပြီး `/verify` ကိုရိုက်ထည့်၍ **key file** တစ်ခုရရန် **console** တွင်ပြထားသော သင်၏ `HWID` ကိုထည့်ပါ

<a href="#"><img width="965" height="415" src="https://media.discordapp.net/attachments/1126893908597669989/1138425541838516294/Untitled.png"></a>
<a href="#"><img width="1324" height="320" src="https://media.discordapp.net/attachments/1126893908597669989/1138426358217842858/Untitled.png"></a>

3. Download the **key file** and put it into the folder ensure that `enc.json` is in the same folder that `v0.x.x.exe`
key file ကို download ပြုလုပ်၍ enc.json အား v0.x.x.exe ရှိသော folder ထဲသို ထည့်ပါ။

4. v0.x.x.exe ကိုထပ်မံ runပါ
5. `GenshinImpact.exe` သိုမဟုတ် `YuanShen.exe` ကိုရွေးချယ်ပါ
6. ဂိမ်းသည် အလိုအလျောက် စတင်လာမည်။ ဂိမ်းမျက်နှာပြင် ပေါ်လာရန်အတွက် ခဏစောင့်ပါ။
7. [Korepi](https://github.com/Korepi/Korepi) GUI ဖွင့်ရန်အတွက် `TAB` key ကို နှိပ်ပါ။

<a href="#"><img width="270" height="200" src="https://images.drivereasy.com/wp-content/uploads/2018/09/img_5ba9fcbbcb694.png"></a>
