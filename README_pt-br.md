## ❓ COMO VERIFICAR E FAZER LOGIN NO DISCORD + COMO OBTER O CARGO FÃS (FANS)

1. Junte-se no [Discord Kitten Buds](https://discord.gg/korek)
2. Você receberá a DM de **Account Verification** e clique para **verify** ou clique em `#account-verify` para **verify**.

<a href="#"><img width="609" height="294" src="https://media.discordapp.net/attachments/1126893908597669989/1144167784201330748/image.png"></a>

3. Mensagem direta para **micah-sign-bot** com `/sign-in`

<a href="#"><img width="329" height="435" src="https://media.discordapp.net/attachments/1116676096360972289/1142149975325814864/image.png"></a>

4. Responda ao bot para ser assinado

<a href="#"><img width="492" height="293" src="https://media.discordapp.net/attachments/1116676096360972289/1142149176189268060/Untitled.png"></a>

5. Você deve fazer isso por 7 dias e, então, receberá uma função de **Fans**. Se você tiver a função **Sponsor** (Patrocinador), só precisará de 3 dias.

## ❓ Para Fans (Fãs) (função de Fans obrigatória)

### Versão

1. Vá até a [página de lançamentos](https://github.com/Cotton-Buds/colorpicker/releases)
2. Faça o download da versão `F` (Ex: [F8](https://github.com/Cotton-Buds/colorpicker/releases/tag/F8))

### Modo de Uso

1. Execute `injector.exe`.
2. Selecione `GenshinImpact.exe` ou `YuanShen.exe`. (Verifique o arquivo `cfg.ini` para ver se o injetor escolheu o caminho correto do jogo)
3. Acesse [nosso discord](https://discord.gg/korek), canal `micah-bot-verify` e digite `/getkey` para obter uma chave.

<a href="#"><img width="300" height="200" src="https://cdn.discordapp.com/attachments/1126893908597669989/1128329159559622676/image.png"></a>
<a href="#"><img width="365" height="164" src="https://media.discordapp.net/attachments/1126893908597669989/1144891137224613969/image.png"></a>

4. Copie a chave que você inseriu no injetor quando ele solicitou.
5. O jogo será iniciado automaticamente. Aguarde até que a interface seja exibida.
6. Pressione `TAB` para abrir a Interface do [Korepi](https://github.com/Korepi/Korepi).

## ❓ Tutorial da Versão do Micah

### Versão

1. Vá até a [página de lançamentos](https://github.com/Cotton-Buds/calculator/releases)
2. Faça o download da versão `v0.x.x` (Ex: [v0.9.7](https://github.com/Cotton-Buds/calculator/releases/tag/v0.9.7))

### Modo de Uso

1. Execute `v0.x.x.exe`.
2. Acesse [nosso discord](https://discord.gg/korek), canal `micah-bot-verify` e digite `/verify` e cole seu `HWID` exibindo no **console** para obter um **arquivo com a key**.

<a href="#"><img width="965" height="415" src="https://media.discordapp.net/attachments/1126893908597669989/1138425541838516294/Untitled.png"></a>
<a href="#"><img width="1324" height="320" src="https://media.discordapp.net/attachments/1126893908597669989/1138426358217842858/Untitled.png"></a>

3. Faça o download do **arquivo com a key** e coloque-o na pasta para garantir que o `enc.json` está na mesma pasta que o `v0.x.x.exe`
4. Execute `v0.x.x.exe` de novo.
5. Selecione `GenshinImpact.exe` ou `YuanShen.exe`. (Verifique o arquivo `cfg.ini` para ver se o injetor escolheu o caminho correto do jogo)
6. O jogo será iniciado automaticamente. Aguarde até que a interface seja exibida.
7. Pressione `TAB` para abrir a Interface do [Korepi](https://github.com/Korepi/Korepi).

<a href="#"><img width="270" height="200" src="https://images.drivereasy.com/wp-content/uploads/2018/09/img_5ba9fcbbcb694.png"></a>
